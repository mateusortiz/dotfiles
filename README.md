# Mateus's Dotfiles

> for more settings in gist

## How to install?

```sh
$ curl -L https://raw.github.com/mateusortiz/dotfiles/master/install.sh || sh
or
$ git clone https://github.com/mateusortiz/dotfiles.git && cd dotfiles && sh install.sh
```
